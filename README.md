# Sample Movie App

iOS / Android Movie Discovery App built with React Native

- \$ yarn add prop-types styled-components @expo/vector-icons axios react-navigation

- \$ yarn start

- \$ expo start

## Screens

- [] Movies
- [] TV shows
- [] Search
- [] Detail

- [x] Now Playing (Movie)
